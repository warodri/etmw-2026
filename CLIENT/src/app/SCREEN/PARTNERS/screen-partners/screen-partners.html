Partner Access with a Private Code

We are building an independent-only audio-first platform where books, essays, and long-form texts become audiobooks. And continue as conversations.

Each audiobook includes a debate space where listeners and authors can discuss the content using voice or text. The goal is simple: help independent ideas be heard, not just published.

What your Partner Code Unlocks

Become a partner and receive a private access code that can be shared with your community.

  • Creators using the code can upload their work for free (no cost to convert books or essays into audiobooks).
  • Readers using the code get their first audiobook for free.

This is designed to lower friction for both writers and listeners during this early phase.

Open to Other Partnership Ideas

This private code is just one way to collaborate.

If you have a different partnership idea (editorial, community-based, experimental, or something else entirely) we are open to discussing it.

The platform is intentionally flexible, and we prefer to shape partnerships together rather than force a fixed model.

Acceso para Partners con Código Privado

Estamos construyendo una plataforma solamente para obras independientes, pensada desde el audio, donde libros, ensayos y textos largos se convierten en audiolibros y continúan como conversaciones.

Cada audiolibro incluye un espacio de debate donde lectores y autores pueden conversar utilizando voz o texto. El objetivo es simple: que las ideas independientes sean escuchadas, no solo publicadas.

Qué Habilita el Código de Partner

Conviértete en partners y recibe un código privado para compartir con tu comunidad.

  • Los creadores que usen el código pueden subir su obra sin costo (no se cobra la conversión a audiolibro).
  • Los lectores que usen el código obtienen su primer audiolibro gratis.

La idea es reducir la fricción tanto para autores como para lectores en esta etapa inicial.

Estamos Abiertos a Otras Formas de Colaboración

El código privado es solo una de las posibles formas de colaboración.

Si tienes otra propuesta — editorial, comunitaria, experimental o distinta — estamos abiertos a conversarla.

Preferimos alianzas flexibles, pensadas en conjunto, en lugar de modelos cerrados.
